AUTHORS: Barry Nwike and Charles Nykamp

DATA: a dataset of chess games which includes the list of moves and metadata like player username and chess opening.

FEATURES CURRENTLY WORKING:
- Retrieving all game data for a particular game id
- Game Viewer page
- Homepage


FEATURES NOT YET WORKING:
- Pagination. Search only returns the first page, no way to switch pages.
- Sorting in the search.


SETUP:
Before running app.py, install a python chess library:
> pip install chess
